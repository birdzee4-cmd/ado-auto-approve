* ชื่อ Tag จะต้องไม่มีเว้นวรรคและไม่มีอักขระพิเศษของ Git

⚠️ Action นี้จะ:

  • สร้าง Git Tag บนระบบ Azure DevOps ภายใต้พาธ refs/tags/<TagName>
  • ผูก Tag เข้ากับ Commit SHA ที่ระบุข้างต้น
  • บันทึก Log การทำ Tag ย้อนหลังลง SharePoint Log เพื่อการตรวจสอบย้อนหลัง
