✅ ยืนยันการ Approve PR

คุณกำลังจะ Approve PR ต่อไปนี้:

⚠️ Action นี้จะ:

  • เปิด Set Auto-Complete ก่อน โดยใช้ Merge (no fast forward)
  • ⭐ Uncheck "Release Notes" optional check (ถ้ามี)
  • เคารพการตั้งค่า completionOptions อื่นที่ผู้เปิด PR ตั้งไว้
  • ส่ง Vote = 10 (Approved) ในนามของ Bot
  • เพิ่ม comment ใน PR ระบุชื่อคุณ
  • บันทึก log ลง SharePoint
  • ไม่ แตะ Work Item / Worklist

❌ ยืนยันการ Reject PR

คุณกำลังจะ Reject PR ต่อไปนี้:

⚠️ Reason จะถูกใส่เป็น comment ใน PR และบันทึกลง log
